
# AI Pipeline Log Summarizer

An AI-powered prototype that summarizes enterprise data pipeline logs into human-readable insights.

## Features
- Detects pipeline failures
- Identifies probable root causes
- Suggests remediation steps
- Simplifies troubleshooting workflows

## Technologies Used
- Python
- Streamlit
- OpenAI
- AI/LLM concepts
- Cloud Data Engineering concepts

## Future Improvements
- Real-time log ingestion
- Databricks integration
- AI-based anomaly detection
- Severity classification
